# WEAL Wellness Tracker - Submission

**Project:** WEAL Wellness Tracker
**GitHub Repository:** [https://github.com/VanshSharmaPES/WEAL_Website_Submission](https://github.com/VanshSharmaPES/WEAL_Website_Submission)

## Overview
A modern, dark-themed health-tech web application for tracking health metrics, nutrition, and psychological states.

## Features
- **Local Storage Integration:** All data is safely stored locally in your browser.
- **CRUD Operations:** Create, Read, Update, and Delete health records instantly.
- **Premium UI/UX:** Styled using design directives for an atmospheric, distinctive look with deep gradients, glassmorphism, and bold Outfit/DM Sans typography.
- **Responsive Design:** Operates smoothly across desktop, tablet, and mobile viewing scenarios.
- **Micro-interactions:** Smooth animations, toast notifications, and modal dialogues.

## Setup Instructions
1. Clone the repository or download the source code.
2. Open `index.html` in any modern web browser.
3. No build tools or dev servers are required.
