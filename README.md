# 🩺 WEAL Wellness Tracker

A clean, modern **Health Wellness Tracker** built with vanilla HTML, CSS, and JavaScript. Demonstrates full **CRUD** (Create, Read, Update, Delete) operations using `localStorage` for persistent data storage.

> Built for **WEAL Health Tech Club** — Technical Recruitment Submission by **Vansh Sharma**.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Create** | Log daily health entries (date, weight, sleep, water, steps, mood, notes) |
| **Read** | View all entries in a sortable data table with search/filter |
| **Update** | Edit any existing entry inline via the form |
| **Delete** | Remove entries with a confirmation modal |
| **Dashboard Stats** | Live averages for sleep, steps, and water intake |
| **Responsive** | Mobile-first, works seamlessly on all screen sizes |
| **Dark Theme** | Modern health-tech aesthetic with teal-green accent |
| **Toast Alerts** | Non-intrusive feedback for all CRUD actions |

---

## 🛠️ Tech Stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, flexbox, animations
- **JavaScript (ES6+)** — IIFE pattern, localStorage CRUD, DOM manipulation
- No frameworks or dependencies.
